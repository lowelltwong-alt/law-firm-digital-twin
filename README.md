# Law Firm Digital Twin

Law Firm Digital Twin is the working repository for the AI-first insurance defense law firm simulation.

Initial scope:

- Build a synthetic, public-data-first simulation of an insurance defense firm.
- Start with labor and employment defense.
- Model the end-to-end lifecycle from intake, conflicts, carrier guidelines, billing, discovery, depositions, mediation, settlement, trial, and post-resolution finance.
- Keep real client data and unclear reuse rights out of the repo until explicit release gates exist.

Current status: repository identity initialized.
