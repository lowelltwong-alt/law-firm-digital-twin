# Source Admission Policy

Every external artifact requires a source-admission receipt before use in a run.

Required fields:

- stable source;
- issuing authority;
- retrieval date;
- effective dates;
- license and terms posture;
- intended use;
- prohibited use;
- source hash or citation;
- normalization method;
- validator;
- owner;
- re-review trigger.

Public does not mean reusable. Unknown provenance is a hold. NITA, Kemper, private carrier guidelines, real client material, case-bound work product, and other third-party copyrighted source text are prohibited unless Lowell later approves a specific license and use receipt.
