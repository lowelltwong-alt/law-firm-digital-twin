# G2 Operational Router Process Boundary

Status: noncanonical qualification; activation remains blocked.

## Decision

Route the synthetic/public operational expert mesh through a closed, provider-neutral JSON protocol executed in a fresh isolated Python subprocess. The worker constructs the fixed G2 employment fixture, matter commitment, registry, and authority ledger. The caller cannot submit a ledger, records, meanings, paths, environment, key, provider, or model. Caller identifiers are syntax-bounded; the public decision uses an opaque request handle and hashes unverified caller-supplied record references.

The worker explicitly does **not** attest its transport. Only the outer qualification harness may report that it observed ten fresh subprocess launches with byte-identical raw responses. Every response binds the request, re-derived decision, fixture, environment, source revision, and exact closed schema.

This remains a same-user subprocess qualification. It proves neither Windows identity/DACL separation nor a protected canonical authority store. It creates no account, directory, ACL, key, pipe, database, backup, external connection, legal authority, or runtime activation.

## Structural gates and kill criteria

- Exact request, decision, receipt, success, and blocked shapes; unknown fields and type confusion fail closed.
- 16 KiB request maximum; 32 identifiers per receipt class; bounded identifier grammar.
- Private/client data, external I/O, human-only effects, invented receipts, and unadmitted authority remain blocked.
- Safe enumerated error codes only; stderr, exceptions, tracebacks, source bodies, sealed data, paths, authority rows, providers, and models are not returned.
- The independent checker re-derives the complete decision and exact receipt fields and rejects false transport, physical-isolation, or activation claims.
- Kill on accepted injection, reflected caller secrets/paths, forged receipt/decision acceptance, raw-response drift, internal leakage, self-attestation, or activation overclaim.
